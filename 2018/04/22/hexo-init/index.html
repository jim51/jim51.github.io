什麼是 Hexo？

先看看官方怎麼說

Hexo 是一個快速、簡單且強大的網誌框架。Hexo 使用 Markdown（或其他渲染引擎）解析您的文章，並在幾秒鐘內，透過漂亮的主題產生靜態檔案。

個人覺得Hexo是個很棒的Blog框架，速度非常快而且支援多種外掛，也能用來寫技術Blog，當然也合適來寫一般Blog(如果不怕學Makrdown的話…)。

安裝

安裝需求

安裝Hexo非常的簡單，但最少還是需要二個軟體(可能你的電腦已經有安裝了)

  • Git
  • Node.js

如果您的電腦剛好已經安裝過上面軟體，只要下npm指令即可完成Hexo安裝。

如果還未安裝也沒有關係。

Windows Git

Git 官網
安裝時基本上都依據預設值就可以了，特別提醒安裝中這個預設值不要動到。
Use Git from the Windows command Prompt

這個動作會在系統加入Git的Path,後面在用cmd或powershell時才用Git指令執行。
安裝完成後，開啟cmd確認一下是否有安裝完成。

Windows Node.js

Node.js 官網
官網的Download for Windows (x64)下面會有二個版本LTS及Current。
這裡會比較建議選擇Current的版本，一般來說效能及功能都會比較強。

安裝的選項都照預設就可以了，等安裝完成執行版本確認。
沒問題就可以進行Hexo安裝了

其他平台 Git

Hexo 官方有提供其他平台的Git安裝說明，請至
https://hexo.io/zh-tw/docs/#%E5%AE%89%E8%A3%9D-Git

其他平倉 Node.js

Hexo 官方有提供其他平台的Node.js安裝說明，請至
https://hexo.io/zh-tw/docs/#%E5%AE%89%E8%A3%9D-Node-js
