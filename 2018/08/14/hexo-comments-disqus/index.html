Blog互動功能是非常必要的，尤其大部份Blog每篇文章下面都很常出現的一個留這版就是Disqus。
現在來說明如何新增這個功能到Hexo。

申請Disqus留言版

這個動作我想是本篇主題最複雜的一步。

申請Disqus登入帳號

只要填入帳號、Email、密碼就可以注冊帳號。如果想用Google、Facebook或是Twitter也可以。

申請網站留言版帳號

在右上方的齒輪點下去，有個[Add Disque to stie]

第二個畫面拉到最下面點選[GET STARTED]

第三個畫面:選擇[I wate to install Disqus on my site]

第四個畫面:設定你Site的相關設定，包含設定Website Name，如果這裡與別人的別名重複會出現
[name]-1

第五個畫面:設定付費，先用免費的有需要可以升級

第六個畫面:選擇設定的網站或系統
因為裡面沒有Hexo可以選擇，所以我們拉到最下面選[Install manually with Universal Code]

第七個畫面:這裡提示你直接使用的碼，但我使用的Theme有支援所以這步完略過就可以了。

其實重點在取得shortname，一般來說shortname是剛設定的Website Name，但如果不確定的話可以參考官方說明
